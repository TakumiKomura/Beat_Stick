# Beat_Stick
Unity Sound Game Jam (2023/8/2 ~ 2023/8/3)に参加した時に作成した「音」をテーマにしたゲームです。
## イライラ棒　×　サウンドゲーム
ステージの上下の壁をゲームのBGMの波形にし、BGMのビートに沿ってステージがビートを刻みます。  
また、プレイヤーはステージ上の五線譜を通り、通った軌跡を元に音を作成します。無事ゴールできるとコースを通った軌跡から作られる音を聞くことができます。  

無事ゴールして素敵な音楽を奏でましょう！
